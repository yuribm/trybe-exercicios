Para que é usado o código de status 400 e 422? 

O código 400 indica "Bad Request", o servidor não conseguiu processar a requisição devido algum erro.

O código 422 indica "Unprocessable Entity", p servidor entende o requisição e que ela está correta, mas não consegue processar.
